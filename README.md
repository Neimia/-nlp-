в этом проекте надо было определить дубликаты по текстовому описанию (данные были в виде таблицы .parquet). идея заключается в том чтобы сравнивать cls токены и какие будут ближе к 0 те более близкие по смыслу то есть дубликату. модель на выходе дает вероятность того что что это дубликаты потому что к последгнему слою применена сигмоида. лосс функция BCELoss (бинарная кросс энтропия) она штрафует сильнее за уверенные не правильные ответы. использовал модель cointegrated/rubert-tiny2 потому что она хорошо работает с русским языком. по итогу получил метрику PRAUC = 0.938
